# Fiverr Clone (Full Stack)

This is a full-stack Fiverr Clone built using React, Node.js, Express, and MongoDB.

## Features

• User Registration & Login  
• JWT Authentication  
• Create Gig  
• View Gigs  
• Profile System  
• Protected Routes  
• MongoDB Database  

## Tech Stack

Frontend:
• React
• Vite
• CSS

Backend:
• Node.js
• Express.js
• MongoDB
• JWT Authentication

## Installation

Backend:

cd backend  
npm install  
npm run dev  

Frontend:

cd frontend  
npm install  
npm run dev
